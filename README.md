## 🚀 项目简介

**选书神器**是一个AI驱动的职场书籍推荐系统，致力于为25-45岁的职场人士提供精准、个性化的阅读推荐。与传统书籍推荐不同，我们专注于职场成长和认知提升领域，提供可解释的精准推荐，帮助用户找到真正适合自己的书籍。

### 🎯 核心特色

- **双维度推荐引擎**：基于职业需求匹配 + 个人偏好匹配
- **可解释性设计**：每本书提供50-150字详细推荐理由
- **学习闭环系统**：推荐→评分→反馈→再推荐的完整链路
- **数据驱动优化**：实时数据看板，持续提升推荐准确度

### 📊 核心指标

- **Pick@1** ≥ 45% (首屏Top1选择率)
- **解释有用率** ≥ 80% (用户认同推荐理由比例)
- **决策时长** ≤ 60s (从推荐到决策的平均时长)

## ✨ 核心功能

### 🎯 精准推荐引擎

#### 双维度匹配算法
- **需求匹配**: 基于职业挑战和具体问题匹配书籍解决方案
- **偏好匹配**: 基于学习风格和阅读偏好推荐合适表达方式
- **冷启动策略**: 新用户通过规则打分获得优质推荐

#### 可解释性设计
每本书推荐包含三个层次：
- **快速了解** (50字): 书籍核心观点
- **为什么适合你** (50字): 结合用户画像的具体理由
- **如何应用** (50字): 两步行动建议

### 🔄 学习闭环系统

```
首推（3-5本） → 用户打分+标签 → 权重调整+再推荐 → 数据看板分析
     ↓                    ↓                    ↓                    ↓
   算法优化         个性化调整           Top3必有变化          持续提升准确率
```

#### 智能反馈机制
- **五星评分**: 整体满意度评估
- **标签系统**: "太理论""很实操""故事强"等具体反馈
- **权重调整**: 基于反馈动态优化推荐算法

### 📊 数据驱动看板

#### 核心指标监控
- **Pick@1**: 首屏Top1选择率 (目标≥45%)
- **解释有用率**: 用户认同推荐理由比例 (目标≥80%)
- **决策时长**: 推荐到决策平均耗时 (目标≤60s)
- **单本均分**: 每本书平均评分 (辅助指标)

#### AB测试支持
- **推荐数量测试**: 3本 vs 5本 vs 10本
- **理由长度测试**: 简洁版 vs 详细版
- **λ参数调优**: MMR多样性平衡测试

### 🎛️ 镜像博主工作台

#### 五旋钮可视化调节
- **难度比例**: 入门 ←→ 专业 (控制推荐书籍难度)
- **故事性**: 理论 ←→ 叙事 (控制表达风格)
- **工具密度**: 思维 ←→ 实操 (控制实用性偏好)
- **时长**: 15分钟 ←→ 600分钟 (控制阅读时间)
- **探索比例**: 保守 ←→ 冒险 (控制推荐多样性)

### 🤖 AI协作开发

#### 推荐算法组合
- **规则打分引擎**: 基于画像的多特征匹配
- **协同过滤引擎**: Surprise库 SVD + KNNBaseline
- **MMR多样性重排**: λ参数平衡准确性vs多样性
- **推荐理由生成**: 模板引擎 + AI增强

### API 接口文档

服务启动后，访问 `http://localhost:8000/docs` 查看完整的API文档。

#### 主要接口

- `POST /api/recommend` - 获取个性化推荐
- `POST /api/feedback` - 提交评分反馈
- `POST /api/rerank` - 基于反馈重新推荐
- `GET /api/dashboard` - 获取数据统计

### 使用示例

#### 获取推荐
```python
import requests

# 用户画像
user_profile = {
    "role": "产品经理",
    "experience": "3年",
    "challenges": ["沟通表达", "团队管理"],
    "learning_style": "实用工具"
}

response = requests.post("http://localhost:8000/api/recommend",
                        json=user_profile)
recommendations = response.json()
```

#### 提交反馈
```python
feedback = {
    "user_id": "user123",
    "book_id": "book456",
    "rating": 5,
    "tags": ["很实操", "案例丰富"]
}

requests.post("http://localhost:8000/api/feedback", json=feedback)
```
