#### **个人特质要求**

```yaml
核心特质:
```

## 🎯 **招聘渠道和策略**

### **推荐招聘渠道**

```yaml
主要渠道:
1. Boss直聘 - 主动搜索和发布岗位
   优势: 候选人活跃，沟通效率高
   策略: 重点筛选有创业公司经验的

2. 拉勾网 - 互联网人才集中
   优势: 候选人质量较高，技术背景匹配
   策略: 重点关注全栈和后端工程师

3. 朋友推荐 - 最可靠的渠道
   优势: 背景可靠，文化匹配度高
   策略: 发动身边朋友推荐合适候选人

4. 技术社区 - 掘金、思否、GitHub
   优势: 技术能力有保证，学习能力强
   策略: 寻找有开源项目或技术文章的候选人
```

### **面试流程设计**

```yaml
第一轮: 简历筛选 + 电话沟通 (30分钟)
- 了解基本技术背景和项目经验
- 沟通薪资期望和工作方式
- 判断是否有创业意愿

第二轮: 技术面试 (90分钟)
- 技术基础问答 (30分钟)
- 代码实现小题 (45分钟) - 实现一个简单的推荐函数
- 技术方案讨论 (15分钟) - 讨论我们的技术架构

第三轮: 综合面试 (60分钟)
- 项目经验深度沟通 (30分钟)
- 产品理解和建议 (20分钟)
- 创业价值观和期望 (10分钟)

决策标准:
- 技术能力: 能独立完成MVP开发
- 学习能力: 能快速适应新技术
- 沟通能力: 能理解产品需求
- 创业匹配: 认同公司愿景
```

## 🚀 **第二阶段：MVP完成后团队配置**

### **团队规模预测**

```yaml
用户规模 → 团队需求:
1000+ 用户: 2人团队 (创始人 + 技术)
5000+ 用户: 3-4人团队 (+ 运营/客服)
10000+ 用户: 5-6人团队 (+ 算法工程师 + 商务)
50000+ 用户: 8-10人团队 (完整功能团队)
```

### **第二阶段团队配置 (3-6个月)**

#### **岗位1: 运营专员/内容运营**

```yaml
主要职责:
- 小红书/知乎等平台内容运营
- 用户社群运营和维护
- 数据分析和用户反馈收集
- 客服和用户支持

技能要求:
- 新媒体运营经验 1-3年
- 文案能力强，了解职场人群
- 数据分析能力，能看懂用户行为
- 沟通能力强，处理用户问题

薪资范围: 8-15K + 绩效奖金
招聘渠道: Boss直聘、拉勾、朋友推荐
```

#### **岗位2: 算法工程师**

```yaml
主要职责:
- 推荐算法优化和迭代
- 用户画像建模和分析
- A/B测试设计和数据分析
- 机器学习模型开发

技能要求:
- 机器学习/推荐系统经验 2-4年
- Python数据科学栈熟练
- 统计学基础，实验设计能力
- 有推荐系统实际项目经验

薪资范围: 25-35K + 期权
招聘渠道: 技术社区、猎头、朋友推荐
```

#### **岗位3: 商务BD/博主运营**

```yaml
主要职责:
- 博主客户开发和维护
- B2B商业模式验证
- 合作伙伴拓展 (出版社等)
- 商业化策略执行

技能要求:
- B2B销售经验 2-5年
- 了解自媒体/KOL行业
- 沟通谈判能力强
- 有一定的产品理解能力

薪资范围: 12-20K + 高绩效提成
招聘渠道: 销售专业平台、行业推荐
```


```yaml
总资金需求: 35-40万元 (6-9个月运营)

资金来源建议:

- 按阶段融资，降低风险
```
