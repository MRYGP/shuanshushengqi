# 运营合伙人招募广告

## 我们在找一个"不安分"的人

**不是招员工，是找合伙人。** **不是现在入职，是为3个月后的爆发做准备。** **不是做执行，是共同定义一个可能改变百万人学习方式的产品。**

------

## 🎯 我们在做什么

我们不是又一个"AI工具"。

我们在重新定义"学习"这件事。

### 当前的问题

你有没有这样的经历：

- 买了10本书，读完的只有2本
- 读完了，但不知道怎么用
- 想学习，但不知道从哪开始
- 每个人都推荐《原则》，但你读不懂

**根本原因：学习不应该是"一刀切"的。**

小学生不能直接学相对论。 工作1年的人不应该直接读《原则》。

但现在所有的推荐系统都在做"匹配"，没人在做"教育"。

### 我们的解决方案

**我们在建设一个"量身定制的递进式成长系统"。**

不只是推荐书，而是：

- 深度理解你的能力水平
- 规划你的6个月成长路径
- 陪伴你从A点到B点
- 动态调整，真正的因材施教

**AI四大革命性功能：**

1. **书本伸缩** - 3分钟到10小时，你决定怎么读
2. **认知翻译** - 把任何书翻译成你能理解的语言
3. **跨书整合** - 知识不再碎片化，而是活的体系
4. **深度匹配** - 发现你从未听过但最适合你的书

**终极愿景：** 不只是读书，而是"体验"。

让你体验马斯克2008年的决策、任正非的至暗时刻。 不是读别人的故事，而是"成为"那个决策者。

从知识到智慧的跃迁。

## 🔥 为什么现在是最好的时机

### AI的窗口期

2024-2025是AI应用的黄金窗口：

- GPT-4/Claude的能力终于够用了
- API成本降低了90%
- 用户习惯了AI对话

**但：**

- 大厂还在做通用工具
- 创业公司还在做Chat包装
- 没人在做真正深度的AI应用

**我们有24个月的窗口期。**

### 市场的空白

```
得到/樊登：做内容，不做系统
ChatGPT：通用工具，不够专业
MBA/EMBA：30万/年，只服务精英

我们：个性化成长系统，999元/年，服务所有人
```

这是一个**百亿级的空白市场**。

### 战略的聪明

我们不会犯Segway的错误（发明了革命性产品，但用户不理解）。

**我们的策略："马车时代不卖汽车"**

```
Year 1-2: 从选书切入（用户立即理解）
Year 3-4: 推出成长系统（用户逐渐接受）
Year 5+: 体验式学习平台（用户已经准备好）
```

就像Netflix：

- 1997: DVD邮寄
- 2007: 流媒体
- 2013: 原创内容

我们用5年时间（不是16年），因为技术已经成熟。

## 💡 我们需要什么样的合伙人

### 不是"运营经理"

如果你想找一份"稳定的工作"，这里不适合你。

如果你只想"完成KPI"，这里不适合你。

如果你把自己定义为"执行者"，这里不适合你。

### 而是"共同创始人"

我们要找的是：

**一个想改变点什么的人。**

不是改变世界（虽然我们有这个野心）。 至少，改变百万人的学习方式。

**一个不满足于现状的人。**

不满足于在大公司做螺丝钉。 不满足于做"还不错"的产品。 不满足于"混口饭吃"。

**一个相信AI能改变教育的人。**

不是因为AI很酷（虽然确实很酷）。 而是真正相信：AI可以让教育更个性化、更高效、更有温度。

**一个想证明自己的人。**

证明你不只是一个"打工人"。 证明你可以从0到1创造一个产品。 证明你有能力成为CEO。

## 🎯 具体的角色定义

### 表面上：运营合伙人（COO）

你的核心职责：

- **用户增长**：从0到1获取种子用户（博主/个人）
- **用户运营**：深度服务，建立口碑
- **数据分析**：找到增长的杠杆点
- **团队管理**：未来带运营团队

### 实际上：不只是运营

在早期创业公司，角色是流动的：

**你可能需要做的：**

- 产品决策：基于用户反馈优化产品
- 商务拓展：和博主、企业谈合作
- 品牌建设：对外讲述我们的故事
- 战略规划：和创始人一起决定公司方向

**甚至：**

- 如果创始人不适合做CEO（可能性存在）
- 你可能成为CEO
- 创始人专注产品

**我们要的是"最强的那个人"，不管Title是什么。**

## 💰 我们能给什么

### 不是高薪

Year 1的工资：10-15K/月（成本价）

如果你需要30K才能生活，这里不适合你。 如果你有房贷车贷压力，这里风险太大。

### 而是股权

**10-15%的股份**（具体看你的能力和加入时间）

假设5年后公司估值30亿（保守）：

- 10% = 3亿
- 稀释后50% = 1.5亿
- 这是你5年的回报

假设5年后公司估值100亿（乐观）：

- 10% = 10亿
- 稀释后50% = 5亿

**但这有风险：**

- 如果公司失败，股份一文不值
- 如果公司只做到1亿估值，10% = 500万（稀释后）

**你需要自己判断这个风险值不值得。**

### 更重要的是成长

5年后，无论公司是否成功：

**你会得到：**

- 从0到1打造产品的经验
- 用户增长的完整方法论
- 带团队的能力
- 和优秀创始人共事的经历
- 创业圈的人脉

**这些是无价的。**

即使公司失败了：

- 你可以再次创业（有经验了）
- 你可以加入其他创业公司（做高管）
- 你可以进大厂（以更高的level）

**5年后的你，远比现在更值钱。**

## ⏰ 时间线（重要）

### 不是现在立即加入

**我们现在的状态：**

- 产品还在设计阶段
- 技术合伙人还在寻找中
- MVP预计3个月后完成

**所以：**

- 现在不需要你全职投入
- 但可以开始深度了解产品
- 参与产品设计讨论
- 为3个月后的爆发做准备

### 理想的时间线

**现在-Month 3（准备期）：**

- 每周2-3小时
- 了解产品和战略
- 给产品建议
- 准备运营计划
- **不占用你的全职工作**

**Month 3（MVP上线）：**

- 如果产品效果好
- 如果我们合作愉快
- **正式作为合伙人全职加入**

**Month 3-6（冷启动期）：**

- 获取前20个种子用户
- 验证产品价值
- 建立增长模型

**Month 6-12（规模化期）：**

- 扩展到100+用户
- 优化增长策略
- 带运营团队

### 为什么这样设计？

**对你的好处：**

- 有3个月了解产品和团队
- 如果不合适，可以随时退出
- 降低你的风险

**对我们的好处：**

- 有3个月了解你的能力和想法
- 确保文化和价值观匹配
- 降低我们的风险

**这是"试婚期"，不是"闪婚"。**

## 🎯 你需要具备什么

### 硬能力（必须有）

```yaml
1. 运营经验：3-5年
   - 不限行业，但教育/SaaS优先
   - 有过0-1冷启动经验
   - 有数据分析能力

2. 用户洞察：能和用户深度对话
   - 不是发问卷，而是真正理解需求
   - 能把用户语言翻译成产品语言
   - 能发现用户自己都没意识到的需求

3. 增长思维：知道如何从0到1
   - 不是砸钱投广告
   - 而是找到增长的杠杆点
   - 能设计增长实验和验证

4. 内容能力：能讲故事
   - 能写文章、案例、软文
   - 能对外讲述产品价值
   - 能打动用户、投资人、媒体
```

### 软能力（更重要）

```yaml
1. 创业心态
   - 把公司当自己的
   - 看到问题主动解决
   - 不需要别人push

2. 学习能力
   - 快速适应新领域
   - 遇到不会的能自己学
   - 不怕挑战

3. 沟通能力
   - 能和创始人坦诚沟通
   - 遇到分歧能理性讨论
   - 不藏着掖着

4. 抗压能力
   - 创业会很苦
   - 会有很多挫折
   - 能不能扛住？

5. 价值观匹配
   - 相信教育的力量
   - 相信AI的潜力
   - 相信小团队能做大事
```

### 加分项（最好有）

```yaml
- 自己就是博主/KOL
- 有产品经理背景
- 有创业经历
- 会一点代码
- 英语好
```
